Data should contain link rows and optionally node rows.
Link rows must have from/to fields or a path field (delimited by "---"). Node rows must have a node field.
Optional link fields: good warn error color width distance speed labelx labely fromside toside tooltip label drilldown
Optional node fields: label labelx labely height width color radius opacity position icon drilldown order tooltip
Drag nodes around to set positions. A button will show to copy positions to the clipboard. Then paste it into the Advanced > Positions formatter field.
